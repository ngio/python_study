# Chart 

## Sunburst Chart
  Sunburst Chart는 계층적 데이터를 시각화하는 데 사용되는 원형 차트입니다. 
  데이터의 루트는 중앙에 있고, 계층적으로 데이터를 표현하며, 내부에서 외부로 확장됩니다. 
  각 섹션은 데이터를 크기별로 구분하여 보여줍니다.

## Pair Plot
  데이터 세트의 숫자 변수 간의 쌍 관계를 보여주는 산점도 그리드입니다. 
  일반적으로 데이터 분포와 변수 간의 관계를 이해하는 데 사용됩니다.
